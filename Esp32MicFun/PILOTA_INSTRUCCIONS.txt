7) Resum tècnic del projecte (Esp32MicFun):
- Plataforma: PlatformIO amb dos entorns:
  - `az-delivery-devkit-v4` (ESP32 real, framework Arduino).
  - `native` (tests locals al host, sense placa).
- Estructura principal:
  - `src/main.cpp`: arrencada del sistema, configuració I2S/pins, Preferences, i creació de tasques FreeRTOS.
  - `src/GlobalDefines.h`: selecció de mida de panell (`PANEL_SIZE_96x54` actiu).
  - `src/Global.h`: estat global, constants d'àudio/FFT, topics MQTT, prefs i handles de tasques/cues.
  - `src/PowerBarsPanel.h`: lògica de mapping XY i utilitats de dibuix del panell LED.
  - `src/FftPower.h/.cpp`: càlcul FFT i extracció de potència per bins (AUTO/PIANO/MATRIX).
  - `src/Effects/*`: renderitzat visual (Wave, LedBars, MatrixFFT, Vert/HorizSpectrogram, Clock, CalcMode, etc.).
  - `src/Tasks/*`: pipeline i serveis en segon pla.
- Flux de dades principal:
  - `ReaderTask`: llegeix I2S del micròfon, normalitza mostres, calcula FFT (`FftPower`) i envia `MsgAudio2Draw` per cua.
  - `DrawerTask`: consumeix la cua, calcula dB per barra, aplica l'efecte segons `DRAW_STYLE` i fa `FastLED.show()`.
  - `WifiReconnectTask`: reconnecta WiFi, OTA, MQTT, NTP i refresc de consum elèctric via HTTP.
  - `ReceiveIRTask`: processa comandaments IR i canvia estils/paràmetres.
  - `VerticalFireTaskPreCalc`: precàlcul del desplaçament vertical per l'efecte fire.
- Integracions:
  - MQTT per control remot i debug (`caseta/spectrometre*` o `caseta/spectrometreBig*` segons panell).
  - OTA via `SharedUtils/OtaUpdater`.
  - IR via `IRremoteESP8266`.
  - FFT via llibreria `tinyu-zhao/FFT`.
- Tests:
  - Fitxer principal: `test/test_fft_power/test_main.cpp`.
  - Tests locals (sense ESP32): `python -m platformio test -e native`.
  - Build de tests ESP32 sense upload: `python -m platformio test -e az-delivery-devkit-v4 --without-uploading --without-testing`.
